# SImple REST CRUD

- Import file *library.sql* to database
- Run program with nodemon
```
npm start
```

## Instal first
```
npm install --save express mysql body-parser
```

- install also nodemon to global
```
npm install -g nodemon
``

- set nodemon in package.json
 ```
 "start": "nodemon app.js"
 ```
 
 - install also env
 ```
 npm install dotenv
 ```
